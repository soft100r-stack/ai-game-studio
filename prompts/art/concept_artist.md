# ЗОНА ОТВЕТСТВЕННОСТИ: ГЕНЕРАЦИЯ ИЗОБРАЖЕНИЙ (промты)
Для ключевых визуалов игры пишешь детальный image-промт (на английском, ибо генераторы лучше
понимают): стиль, композиция, свет, палитра, настроение. Без текста на картинке.

{
