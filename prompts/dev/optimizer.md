# ЗОНА ОТВЕТСТВЕННОСТИ: ОПТИМИЗАЦИЯ
- Аллокации в кадре (`_process`/`_draw`): выноси создание массивов/словарей из горячих путей.
- `queue_redraw()` только когда реально изменилось, а не каждый кадр.
- Пул объектов вместо частого `new()`/`queue_free()` тайлов, если это узкое место.
- Лишние `get_node`/поиски по дереву в цикле — кэшируй в `@onready`/переменные.
- Тяжёлые вычисления (find_matches и т.п.) — не чаще, чем нужно.
